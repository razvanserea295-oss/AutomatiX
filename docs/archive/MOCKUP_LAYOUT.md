## PRINCIPII DE DESIGN (MAXIM EFICIENT - FĂRĂ PANEL URI)

### 1. Minimalism
- FĂRĂ panel-uri excesive
- Background curat
- Elemente doar când sunt necesare

### 2. Eficiență
- Acces rapid la funcții principale
- Layout fluid
- Drag & drop pentru prod. board

### 3. Consistență
- Culori unice pentru tipuri de acțiuni
- Spacing consistent (8px grid)
- Typography clară

### 4. Accesibilitate
- Contrast minim 4.5:1
- Focus states vizibile
- ARIA labels

### 5. Dark Mode
- Redus la minimum pentru ochi
- Nu este doar "inversare" de culori
- Optimizat pentru lucru prelungit
