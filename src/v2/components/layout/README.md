# Layout templates

Canonical page shells — use these instead of inventing per-page grids.

| Template | Use for |
|----------|---------|
| `PageChrome` | Title + subtitle + actions (+ optional toolbar) |
| `Panel` | Bordered surface with header band + scroll body |
| `ListPageLayout` | Single-column list/table |
| `MasterDetailLayout` | 7/5 or 8/4 split |
| `DashboardLayout` | KPI strip + main + footer |

Spacing: `px-6`, `gap-6`, 8px grid only.

Migrated: `ReportsPage`, `UserSessionsPage`.
